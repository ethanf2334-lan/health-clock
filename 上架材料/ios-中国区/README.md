# 健康时钟 iOS 中国区上架材料包

这个目录用于集中存放健康时钟首版 iOS 上架材料。

## 文件说明

- `00-上线步骤.md`：从开发者后台、App Store Connect、构建上传到提交审核的完整步骤。
- `01-AppStoreConnect信息.md`：App 名称、描述、关键词、隐私问答等可复制材料。
- `02-隐私政策.md`：可发布到官网或 App 内的隐私政策草稿。
- `03-用户协议.md`：用户协议草稿。
- `04-审核备注与测试账号.md`：提交审核时给 Apple 审核员的说明模板。
- `05-截图与素材清单.md`：截图、图标、App 预览视频准备清单。
- `06-中国区合规与备案清单.md`：ICP/APP 备案和中国大陆上架注意事项。

## 必须补齐

- 开发者姓名、联系邮箱、通信地址。
- 隐私政策 URL、技术支持 URL。
- 正式后端域名和备案信息。
- 审核测试账号和固定验证码。
- 产品截图和 1024 x 1024 App 图标。

## 当前项目关键信息

- App 中文名：健康时钟
- 当前 iOS 显示名：Health Clock
- Bundle ID：`com.lanxf.healthClock`
- Flutter 版本号：`1.0.0+1`
