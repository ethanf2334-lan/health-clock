## 描述

健康时钟是一款面向个人与家庭的健康提醒和健康档案管理 App。

你可以用自然语言快速创建健康提醒，例如用药、复诊、体检、测血压、测血糖等日常安排；也可以为家人建立独立档案，集中管理健康事件、医疗文档和指标记录。

主要功能：

- AI 辅助创建健康提醒：输入一句话，整理出待确认的提醒草稿。
- 家庭成员档案：为自己、父母、孩子等成员分别管理健康事项。
- 日历与提醒：按日期查看待办事项，减少漏记和遗忘。
- 医疗文档管理：上传体检报告、化验单等文档，便于后续查找。
- OCR 识别与结构化整理：从文档图片中提取文字，辅助生成健康记录。
- 健康指标记录：记录血压、血糖、体重等指标，沉淀长期健康数据。

健康时钟只提供健康记录、提醒和信息整理服务，不提供医学诊断、治疗建议或处方服务。任何健康问题请咨询合格医疗专业人士。

## App 隐私问答建议

请按实际 SDK 和后端日志最终核对。以下基于当前代码和产品文档推断：

### 可能收集的数据

- 联系信息：手机号（用于登录/账号识别）
- 用户 ID：Apple 登录标识、用户账号 ID
- 健康与健身：健康提醒、健康指标、家庭成员健康档案
- 敏感信息：用户上传的医疗文档可能包含敏感健康信息
- 照片或视频：用户选择/上传的医疗文档图片
- 音频数据：如正式启用语音输入，需要披露语音数据；如果首版不启用语音输入，应移除麦克风/语音权限或说明不收集
- 诊断数据：如接入崩溃分析、性能日志，需要披露

### 数据用途

- App 功能：账号登录、提醒、档案、文档识别、健康指标记录
- 分析：仅当接入统计分析 SDK 时选择
- 产品个性化：仅当用数据个性化推荐或排序时选择
- 开发者广告或第三方广告：当前不建议选择
- 跟踪：当前不应选择，除非接入跨 App/网站追踪或广告归因 SDK

### 是否与用户关联

健康数据、文档、手机号、账号 ID 通常与用户账号关联，应选择“与用户关联”。

### 是否用于追踪

当前产品定位不需要追踪。不要接入广告追踪 SDK；如无 ATT 场景，应回答“不用于追踪”。
