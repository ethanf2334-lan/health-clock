# 健康时钟 MVP 项目总结

## 项目概述

健康时钟是一款基于 AI 的家庭健康提醒与健康档案管理 App，采用 Flutter + FastAPI 技术栈，iOS 首发。

- 开发启动时间：2026-04-23
- 当前时间点：2026-04-25
- 当前状态：MVP 核心链路已基本可体验，正在补齐 P2 体验优化
- 进度参考：详见 `docs/健康时钟-MVP项目进度总览.md`

## 当前结论

项目已经从“能力搭建”推进到“主流程可跑通”阶段：

- 后端核心能力完整度较高，登录、成员、提醒、文档/OCR、健康指标等 API 均已落地。
- 前端已接通主要业务链路，支持手机号验证码登录、成员管理、提醒管理、AI 创建提醒、文档上传/OCR、健康指标和本地通知。
- 第六轮补齐了健康日历的周/月网格视图，并修复了文档详情页导致 Flutter 测试无法编译的语法问题。

## 技术架构

### 前端

- 框架：Flutter 3.0+
- 状态管理：Riverpod
- 路由：GoRouter
- 网络请求：Dio
- 图表：fl_chart
- 文件选择：file_picker / image_picker
- 本地通知：flutter_local_notifications

### 后端

- 框架：FastAPI
- 认证：阿里云短信验证码 + 自签 Supabase 兼容 JWT
- 数据库：Supabase Postgres
- 对象存储：Cloudflare R2
- AI 框架：LangChain
- AI 模型：Dashscope Qwen
- OCR：百度 OCR

- Home 四 Tab 架构
