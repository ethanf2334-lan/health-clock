# 健康时钟 MVP 项目总结

## 项目概述

健康时钟是一款基于 AI 的家庭健康提醒与健康档案管理 App，采用 Flutter + FastAPI 技术栈，iOS 首发。

**开发周期**: 2026-04-23 开始
**当前状态**: MVP 后端完成，前端框架搭建完成
**完成度**: 后端 100%，前端 30%，文档 100%

## 技术架构

### 前端
- **框架**: Flutter 3.0+
- **状态管理**: Riverpod
- **路由**: GoRouter
- **本地数据库**: Drift
- **网络请求**: Dio
- **认证**: Supabase Flutter SDK

### 后端
- **框架**: FastAPI
- **认证**: Supabase Auth (JWT)
- **数据库**: Supabase Postgres
- **对象存储**: Cloudflare R2
- **AI 框架**: LangChain
- **AI 模型**: Dashscope Qwen3.6-Plus
- **OCR**: 腾讯云 OCR
