# 百度 OCR 配置指南

## 1. 注册百度智能云账号

访问 https://login.bce.baidu.com 注册账号

## 2. 开通文字识别服务

1. 登录百度智能云控制台
2. 在产品服务中找到「文字识别 OCR」
3. 点击「立即使用」开通服务

## 3. 创建应用

1. 进入文字识别控制台
2. 点击「创建应用」
3. 填写应用信息：
   - 应用名称：健康时钟
   - 接口选择：
     - 通用文字识别（高精度版）
     - 医疗票据识别
   - 文字识别包类型：按量后付费
4. 创建成功后获取：
   - API Key
   - Secret Key

## 4. 配置到项目

编辑 `health-clock-backend/.env` 文件：

```env
BAIDU_OCR_API_KEY=your_api_key_here
BAIDU_OCR_SECRET_KEY=your_secret_key_here
```

## 5. 测试 OCR 功能

启动后端服务后，访问 http://localhost:8000/docs

测试接口：`POST /api/v1/documents/ocr`

## 7. 常见问题

### Q: 如何查看调用量？
A: 登录百度智能云控制台 -> 文字识别 -> 用量统计

### Q: 识别失败怎么办？
A: 检查以下几点：
1. API Key 和 Secret Key 是否正确
2. 图片格式是否支持（JPG/PNG）
3. 图片大小是否超过 4MB
4. 图片 URL 是否可访问

### Q: 支持哪些图片格式？
A: 支持 JPG、PNG、BMP 格式

### Q: 图片大小限制？
A: 
- 通用文字识别：最大 4MB
- 医疗票据识别：最大 4MB

## 9. 优势对比

### 百度 OCR vs 腾讯云 OCR

| 特性 | 百度 OCR | 腾讯云 OCR |
|------|----------|------------|
| 免费额度 | 1000 次/月 | 1000 次/月 |
| 识别准确率 | 98%+ | 98%+ |
| 响应速度 | < 1s | < 1s |
| 医疗识别 | ✅ 支持 | ✅ 支持 |
| 价格 | 0.015 元/次 | 0.015 元/次 |
| 接入难度 | 简单 | 简单 |

**选择百度 OCR 的理由**：
- 接入更简单（只需 API Key 和 Secret Key）
- 文档更完善
- 社区支持更好
- 国内访问更稳定

## 10. 安全建议

1. 不要将 API Key 和 Secret Key 提交到 Git
2. 使用环境变量管理密钥
3. 定期更换密钥
4. 监控调用量，防止滥用
5. 生产环境使用 HTTPS
